# wenlock2

c++ discord bot

PhoneDB Scraper and Captcha Verification

Captcha depends on [wenlock2-server](https://github.com/ExposureMG/wenlock2-server)

## Dependencies

- D++
- Libxml2 (XML/HTML Parsing)
- Libcurl (REST)
- SQLite3


## Download / Deploy

Docker images available [here](https://hub.docker.com/repository/docker/exposuremg9936/wenlock2)

Designed to be used with [wenlock2-compose](https://github.com/ExposureMG/wenlock2-compose)
